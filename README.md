# G26_Site - Portafolio de Desarrollo Web

Este repositorio contiene mi sitio web portafolio profesional como Tarea del curso G26 de https://www.instagram.com/softwarelibrechile/, diseñado para presentar mis proyectos, trayectoria y habilidades técnicas. El sitio ha sido desarrollado aplicando buenas prácticas de frontend, con un enfoque en la mantenibilidad del código y un diseño responsive.

🚀 **Ver sitio en vivo:** https://ipe-s.github.io/G26_Site/

---

## 🛠️ Tecnologías y Metodologías

Para este proyecto se han implementado las siguientes herramientas:

*   **HTML5:** Estructura semántica avanzada para SEO y accesibilidad.
*   **Sass (SCSS):** Uso de variables, anidamiento (nesting) y mixins para optimizar la hoja de estilos.
*   **Metodología BEM:** Arquitectura de clases (Block, Element, Modifier) para un CSS modular.
*   **Google Fonts:** Integración de tipografía personalizada (Roboto).
*   **Material Symbols:** Iconografía moderna para una interfaz intuitiva.
*   **Git & GitHub:** Control de versiones y despliegue automatizado.

## 🏗️ Arquitectura de Archivos

El proyecto mantiene una separación clara entre los archivos de desarrollo y los de producción:

```text
/
├── css/                # Archivos CSS compilados (interpretados por el navegador)
│   └── style.css
├── styles/             # Código fuente original en Sass (SCSS)
│   └── index.scss
├── img/                # Recursos gráficos y activos visuales
├── index.html          # Punto de entrada principal del sitio
└── README.md           # Documentación del proyecto
